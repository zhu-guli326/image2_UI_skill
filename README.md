# Image to UI Skill

将 UI 截图、设计稿、图片快速转换为可实现的前端代码和图片资产的 AI Agent Skill。

## 功能介绍

你有没有遇到过这样的场景：
- 看到一张漂亮的 UI 设计图，想要快速实现成代码
- 需要从一张设计稿中提取出哪些部分应该用代码实现，哪些需要生成图片
- 想把产品图片做成透明背景的抠图资产

这个 Skill 可以帮你：
1. **分析 UI 图片** - 智能识别哪些元素适合用代码实现，哪些需要生成图片
2. **生成资产清单** - 输出需要生成的图片资产、尺寸、提示词
3. **生成图片** - 帮你创建首屏背景、插画、抠图等位图资产
4. **集成到代码** - 将生成的图片和代码整合到你的前端项目中
5. **审查验收** - 自动检查还原度，确保最终效果接近原图

## 核心能力

### 智能分析
- 识别可代码渲染的元素（按钮、卡片、导航、表单）
- 识别需要生成的图片资产（插画、照片、复杂背景）
- 识别需要抠图的素材（人物、产品透明背景）

### 完整流程
```
上传 UI 参考图 → 前期审查 → 资产规划 → 生成图片 → 集成代码 → 页面审查
```

### 格式支持
- 支持 Figma 导出图、设计稿截图、App UI 截图
- 支持桌面端和移动端响应式分析
- 支持 PNG、WebP、JPG 等图片格式

## 使用场景

- **设计师** - 快速验证设计稿能否实现，获取开发建议
- **前端开发者** - 将 UI 参考图转成可执行的代码方案
- **产品经理** - 快速搭建可交互的原型页面
- **创业者** - 把灵感图快速变成可演示的产品原型

## 使用方法

### 在 Cursor 中使用

1. 确保你的 Cursor 项目中已安装此 Skill
2. 上传你的 UI 参考图
3. 描述你的需求，例如：
   - "帮我分析这个设计稿"
   - "把这个 UI 转成代码"
   - "提取页面中的图片资产"

### Skill 会自动

- 分析 UI 元素并分类
- 生成前期审查文档
- 输出资产清单和生成提示词
- 生成需要的图片资产
- 编写可运行的代码
- 验证最终效果

## 示例工作流

### 1. 上传 UI 参考图

将你希望实现的 UI 图片发送给 AI。

### 2. 获取前期审查

Skill 会分析并返回：
- 哪些元素容易实现
- 哪些元素需要生成图片
- 哪些元素需要抠图处理
- 需要确认的问题

### 3. 生成资产和代码

确认后，Skill 会：
- 生成首屏背景图
- 生成插画/图标
- 提取抠图素材
- 编写前端代码（React/Vue/HTML）

### 4. 审查与调整

- 自动检查还原度
- 标注与原图的差异
- 调整直到满意

## 技术细节

### 输出内容

- **前期审查文档** - 分析报告，包含难度评级和实现建议
- **资产清单** - 图片资产列表，包含尺寸、提示词、导出格式
- **前端代码** - 可运行的 HTML/CSS/React/Vue 代码
- **生成资产** - PNG、WebP 格式的图片文件

### 响应式支持

- 自动识别桌面端和移动端布局
- 生成不同断点的图片
- 确保各端显示效果一致

### 可访问性

- 所有 UI 文字保留为代码
- 生成合适的 alt 文本
- 支持屏幕阅读器

## 注意事项

- 复杂 Logo 和商标建议使用原图
- 特殊字体需要确认授权
- 生成的图片风格可能与参考略有差异
- 建议先在小范围测试后再全量使用
